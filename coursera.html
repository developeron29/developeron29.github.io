<div style="position:fixed;top:10px;right:30px;z-index:1000"> <a href="./index.html"><button type="button" class="btn btn-link btn-lg"> Home</button></a> </div> # [Coursera preview downloader](https://github.com/developeron29/coursera-preview-download) Coursera is an educational company that offers Massively offered Online courses(MOOC) founded by computer science professors Andrew Ng and Daphne Koller from Stanford University. Some courses on coursera have their preview available if these courses have been offered before but they don't have any downloadable link. So to download these videos on your system for offline reference ,The script on my git repository could be useful. ##Usage ```shell ##Shoot the terminal- ##For mac users who **don't** have `wget` installed ##Run ##To install homebrew - ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)" ##To install wget - brew install wget brew install lynx ##To install the Cousera preview videos- sh coursera-preview-download.sh https://class.coursera.org/<course>/lecture/preview ##example:- sh coursera-preview-download.sh https://class.coursera.org/nlp/lecture/preview ``` ####FeedBack and Bugs are welcome
